# AL MAIRAJ MOTORS

A modern car dealership website showcasing new and used cars in Pakistan.

## Features

- Responsive design
- Car gallery with image navigation
- New and used car listings
- Contact form
- WhatsApp integration
- Modern UI with animations

## Project Structure

```
├── html/          # HTML pages
├── css/           # Stylesheets
├── js/            # JavaScript files
├── images/        # Car images and assets
└── index.html     # Main entry point
```

## Getting Started

### Local Development

1. Install Node.js (if not already installed)
2. Install http-server globally:
   ```bash
   npm install -g http-server
   ```
3. Run the server:
   ```bash
   http-server -p 8080
   ```
4. Open your browser and navigate to `http://localhost:8080`

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Particles.js (for background animations)
